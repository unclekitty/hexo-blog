CSS 基础框盒模型介绍

Kitty叔叔
2019-07-15

该模型定义了 CSS 如何对元素的各个组成进行布局，这些组成包括他们的内容 Content、内边距 Padding、边框 Border，和外边距 Margin。

标准盒模型

width = contentWidth;

height = contentHeight;

IE 盒模型

width = contentWidth + paddingLeft + paddingRight + borderLeftWidth + borderRightWidth

height = contentHeight + paddingTop + paddingBottom + borderTopWidth + borderBottomWidth

通过 CSS 中box-sizing这个属性

  • content-box：默认值，标准盒模型
  • border-box: IE 盒模型
  • inherit: 从父元素继承
