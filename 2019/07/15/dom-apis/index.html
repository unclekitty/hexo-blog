常用 DOM API

Kitty叔叔
2019-07-15

查找节点

根据 ID 查找元素，大小写敏感，如果有多个结果，只返回第一个

根据类名查找元素，多个类名用空格分隔，返回一个 HTMLCollection

根据标签查找元素(* 表示查询所有标签), 返回一个 HTMLCollection

根据元素的 name 属性查找，返回一个 NodeList

根据 CSS 选择器查找元素， 如果匹配到多个结果，只返回第一个

根据 CSS 选择器查找元素， 如果匹配到多个结果，返回一个 NodeList

创建节点

创建元素

创建文本元素

createDocumentFragment

克隆节点

node.cloneNode(true/false), 它接收一个 bool 参数，用来表示是否复制子元素

插入节点

appendChild

insertBefore

insertAdjacentHTML

position 表示插入内容相对于元素的位置 ()

  • ‘beforebegin’：元素自身的前面。
  • ‘afterbegin’：插入元素内部的第一个子节点之前。
  • ‘beforeend’：插入元素内部的最后一个子节点之后。
  • ‘afterend’：元素自身的后面。

text 是要被解析为 HTML 或 XML 元素，并插入到 DOM 树中的 DOMString

删除节点

removeChild 删除指定的子节点并返回子节点

替换节点

replaceChild 用于将一个节点替换另一个节点

属性操作

setAttribute 给元素设置属性

getAttribute 返回指定的特性名相应的特性值

window.getComputedStyle 获取应用到元素上的所有样式

getBoundingClientRect 返回元素的大小以及相对于浏览器可视窗口的位置

clientRect 是一个 DOMRect 对象，包含 width、height、left、top、right、bottom
